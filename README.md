# philosophers
42 philosophers

This project is an introduction about thread.

The programme take in argument `number of philosophers` `time to die` `time to eat` `time to sleep` `(optional) amount of time each philosophers must eat`

## makefile
- **`make`**: Compiles the project.
- **`make clean`**: Removes the `objet` folder.
- **`make fclean`**: Runs `make clean` and removes the executable and the `libft` library.
- **`make re`**: Runs `make fclean` and then `make`.
